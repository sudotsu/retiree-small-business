## Content Organization Rules

### File Naming Convention
- **Skeletons:** `RBB_Module-XX_[Topic]_v2.md`
- **Workbooks:** `RBB_Module-XX_Workbook_v1.md`
- **Slides:** `RBB_Module-XX_Slides_v1.md`
- **Instructor:** `RBB_Module-XX_Instructor_v1.md`
- **Hooks:** `RBB_Module-XX_Hooks_v1.md`

### Module Structure (Each Module Should Have)
1. **Skeleton** - Full lesson plan, outcomes, activities
2. **Workbook** - Student worksheets (printable + fillable)
3. **Slides** - Presentation deck outline/notes
4. **Instructor Guide** - Facilitation notes, timing, examples
5. **Marketing Hooks** - Module-specific marketing copy

### Cross-Reference Check
- Module X should reference Module X-1 (previous) and Module X+1 (next)
- All modules should align with Essentials track
- All modules should reference quality/7-Step-Validation
