# Phase 5: Admin Panel (Gate 5)

**Goal:** Non-technical user can manage course content

**Key Tasks:**
1. Admin authentication (role-based)
2. Course/module/lesson CRUD
3. Video upload or embed URL
4. Worksheet PDF upload
5. Quiz builder
6. Publish/unpublish controls
7. User management

**Validation:**
- Admin can log in
- Can create new lesson without code
- Can upload video/worksheet
- Changes appear in student portal
- No coding required for content updates

**Estimated Time:** 6-8 hours

Consider using admin UI library (e.g., Refine, React Admin).
