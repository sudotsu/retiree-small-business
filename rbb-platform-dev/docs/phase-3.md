# Phase 3: Course Viewing (Gate 3)

**Goal:** Students can watch videos, track progress, navigate lessons

**Key Tasks:**
1. Build student portal dashboard
2. Course overview page (modules/lessons list)
3. Lesson viewer with video player
4. Mark lesson complete functionality
5. Downloadable worksheets
6. Lesson prerequisites/locking
7. Prev/next navigation

**Validation:**
- Enrolled student sees course in portal
- Can watch video lesson
- Can download PDF worksheet
- Progress persists to database
- Cannot access locked lessons
- Navigation works correctly

**Estimated Time:** 5-6 hours

See API_SPEC.md for lesson_progress queries.
